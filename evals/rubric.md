# 人工质量评分量表

每个用例由至少一名短剧制作成员评分。机器检查通过不等于内容可用，人工评分用于判断表演设计是否真的减少制作成本。

## 五个维度

每项使用 1-5 分：

| 分数 | 定义 |
|---|---|
| 1 | 不可用，核心方向错误，需要重做 |
| 2 | 大改后可用，存在明显人物或执行问题 |
| 3 | 可用但需多处修改，能保留约一半内容 |
| 4 | 轻改可用，人物与动作基本准确 |
| 5 | 可直接使用，无实质修改 |

对以下维度分别评分：

1. **人物一致性**：动作、Tell、关系反应是否符合人物档案。
2. **情绪节拍**：心理动作到物理动作的推进是否清楚，是否存在突兀跳变。
3. **动作可执行性**：Seedance 能否看见并执行描述，是否避免抽象形容词。
4. **对白自然度**：目标市场语言是否自然，语气、音量、停顿是否支持潜台词。
5. **首稿可用性**：在真实制作中是否愿意直接使用或仅做轻改。

## 必填记录

- `采用判断`：直接使用 / 轻改使用 / 大改使用 / 放弃。
- `修改字符数`：最终版本相对模型原稿的新增、删除和替换字符总数。
- `原稿字符数`：模型 Prompt 字符数。
- `修改比例`：`修改字符数 / 原稿字符数`。
- `首要失败原因`：人物、节拍、动作、对白、格式、模型错误或其他。

## P1 通过门槛

- 机器格式与语言硬门槛通过率 100%。
- “直接使用 + 轻改使用”占比不少于 80%。
- 可用结果的修改比例中位数不高于 20%。
- 五个评分维度的平均分均不低于 4.0。
- P50 端到端耗时不高于 30 秒，P95 不高于 60 秒。
