# 结构化动作与物理因果

## 使用边界

本规则服务于 Seedance 2.0 文戏和情感戏。它把已有的“可观测，不写意”进一步组织成稳定的描述顺序，但不要求每个镜头机械填满所有槽位，也不替代人物动机、节拍和对白设计。

社交媒体中关于“稳定性提升 30%”或“成功率提升 15-25%”的数字没有项目内可复核实验，只能作为待验证假设，不得写成 Skill 事实。

## 镜头内默认顺序

默认顺序：主体 -> 主体动作/状态 -> 场景环境及物理反馈 -> 镜头语言 -> 氛围/风格。

1. **主体**：先说清谁在画面中、站位或与道具的关系，避免先写氛围或运镜后才出现人物。
2. **动作/状态**：用镜头可记录的位移、肌肉、呼吸和声音，不写“她在思考”“他很生气”“她很优雅”。
3. **环境反馈**：只有环境确实影响主体或道具时才写，必须包含原因与可见结果。
4. **镜头语言**：每个镜头最多一个主要运动；人物动作复杂时优先固定镜头。
5. **氛围/风格**：放在可执行信息之后，用具体光线、色温、声音或空间状态承载，不用“高级感”“电影感”。

## 动作从大到小

动作层级：整体姿态/重心 -> 头部动作 -> 眼神/表情 -> 手部动作/道具 -> 细节反馈。

- **整体姿态/重心**：站、坐、侧身、前倾、后撤、肩背方向、重心落点。
- **头部动作**：低头、抬下巴、偏头、转头的方向和幅度。
- **眼神/表情**：目光落点、眼睑、眉、鼻翼、嘴唇、下颌等局部信号。
- **手部动作/道具**：哪只手、哪根手指、接触什么、动作如何开始和停止。
- **细节反馈**：衣料褶皱、皮肤受压、杯中水纹、纸张抖动、呼吸带来的胸肩变化。

这是组织信息的优先顺序，不是绝对的时间顺序。若剧情重点是“手先碰到旧物，脸随后失控”，应保留真实因果，不得为了格式强行把面部反应提前。

## 物理因果模板

只写与原场景一致、镜头可见的原因和结果。不要为了“真实感”凭空增加天气、烟雾或复杂光效。

| 原因 | 必须出现的可见结果 | 示例 |
|---|---|---|
| 风/气流 | 头发、衣摆、树叶、纸张发生方向明确的摆动或贴附 | 风从画面右侧吹来，湿发贴到她左侧脸颊，她抬手拨开。 |
| 雨/水滴 | 伞面、衣物、地面或皮肤出现溅起、汇聚、滑落、湿润或反光 | 雨点击中伞面，水花沿伞沿溅起；鞋底踏入积水，涟漪向外扩散。 |
| 雾/水汽 | 轮廓、玻璃或光束出现漫散、凝结、遮挡或逐渐显露 | 水汽凝在镜面，她擦过后留下清晰掌痕，半张脸从雾后显露。 |
| 阳光/逆光/灯光 | 人物、道具或空间出现光斑、轮廓、投影、反射或明暗边界 | 阳光穿过百叶窗，在她握紧的手背投下移动光斑。 |
| 接触/受力 | 皮肤、衣料、纸张、液体或道具出现压痕、褶皱、位移、震动 | 指尖压住信纸边缘，纸张先弯曲再轻微回弹。 |

## 正反示例

**错误：** 女生在雨中走路，撑着伞，神情悲伤。

**正确：** 女生肩膀内收，左手握住伞柄，目光停在前方路口。雨点击中伞面，细小水花沿伞沿溅起；风把湿发贴到她的脸颊，她用右手拇指擦去下颌的水珠。镜头固定在中近景，环境声只保留雨点和鞋底踏水声。

**错误：** 她优雅地坐着思考。

**正确：** 她侧身坐在椅子前缘，重心落在左侧髋部；下巴微收，目光停在画面外。左手撑住膝盖，食指缓慢摩挲裙料，衣料在指腹下形成一道短褶皱。

## 固定负向段

使用以下完整文案作为提示词最后一段：

```text
负向约束：避免画面抖动、镜头剧烈晃动、人物面部变形、肢体错乱、手指数量异常、画面闪烁、过曝、过暗、字幕、水印和低画质模糊；不要快速切换、戏剧化大动作和生硬切镜。
```

负向段只负责降低常见生成故障，不能替代正向动作描述。优先写清主体、动作和因果，再追加该段。
